This peer's status #(peerStatus)# Virgin - You have not yet published your peer, because you have not yet used the proxy. With this status you are not allowed to search other peers. :: Junior - You cannot be reached. A possible reason is that you are behind a firewall, NAT or Router. But you can search the internet using the other peers' global index on your own search page. Please be fair, contribute your own index to the global index. :: Senior - You are running a server and you support the global internet index, which you can also search yourself.
